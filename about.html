About Aviation Data Explorer

A comprehensive tool for aviation enthusiasts, pilots, and professionals.

Our Mission

We aim to provide easy access to aviation data that is typically scattered across multiple sources. Our platform consolidates airport information, charts, routes, and live virtual flight data in one convenient location.

Data Sources

Our primary data source is the AviationAPI, which provides comprehensive information about airports, charts, and preferred routes. We also integrate with VATSIM for real-time virtual flight data.

Features

  • Search airports by ICAO/FAA code
  • Access airport charts and diagrams
  • View preferred routes between airports
  • Monitor real-time VATSIM flights and controllers
  • Check current weather (METAR) and forecasts (TAF)
